# 变形

## `tanslate(x, y)`移动

| 说明 | 移动canvas的原点到一个（x,y)位置         |
| ---- | ---------------------------------------- |
| 参数 | x->水平方向的偏移量，y->垂直方向的偏移量 |



## `rolate(deg)`旋转

deg: 弧度

| 说明 | 以canvas的原点为定点旋转 |
| ---- | ------------------------ |
| deg  | 旋转的弧度               |

## `scale(x, y)`缩放

| 说明 | 对形状、位置进行缩小和放大 |
| ---- | -------------------------- |
| 参数 | 水平和垂直方向的缩放因子   |

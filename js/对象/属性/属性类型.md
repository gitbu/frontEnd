# 属性类型

属性类型分为两种

> 数据属性

| 属性           | 默认值       | 解释                               |
| ------------ | --------- | -------------------------------- |
| value        | undefined | 属性对应的值                           |
| configurable | true      | 是否可以删除、是否可以再次修改数据的属性configurable |
| enumerable   | true      | for in 是否可便利                     |
| writable     | true      | 是否可写                             |



> 访问器属性

| 属性   | 默认值       | 解释        |
| ---- | --------- | --------- |
| set  | undefined | 设置属性值时会调用 |
| get  | undefined | 访问属性值是会调用 |

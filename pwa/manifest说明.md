# manifest说明

```json
{
  "name": "PWA Manifest Demo",
  "short_name": "Manifest Demo",
  "start_url": "./index.html",
  "theme_color": "#4374A5",
  "background_color": "#4374A5",
  "display": "standalone",
  "orientation": "natural",
  "icons": [{
    "src": "images/launcher-icon.png",
    "sizes": "192x192",
    "type": "image/png"
  }]
}
```

* name: 应用名称
* short_name：应用短名称，使用场景为
* start_url：启动应用时加载的 URL。
* theme_color：工具栏的色值
